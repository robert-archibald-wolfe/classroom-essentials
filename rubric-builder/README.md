# Rubric Builder - Google Sheets Add-on

Create and apply grading rubrics.

**Status:** Coming Soon

## Features (Planned)

- Build rubrics with criteria and levels
- Point values or descriptive scales
- Apply rubric to student work
- Calculate total scores
- Export formatted rubrics
- Rubric templates library
- Copy feedback comments quickly
