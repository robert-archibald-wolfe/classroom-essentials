# Parking Lot - Google Slides Add-on

Digital questions board for during lessons.

**Status:** Coming Soon

## Features (Planned)

- Students submit questions without interrupting
- Questions display in sidebar or overlay
- Teacher can address questions when ready
- Mark questions as answered
- Export questions for follow-up
- Anonymous or named submissions
