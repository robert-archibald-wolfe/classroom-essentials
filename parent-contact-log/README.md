# Parent Contact Log - Google Sheets Add-on

Track parent and guardian communications.

**Status:** Coming Soon

## Features (Planned)

- Log phone calls, emails, conferences
- Link to student roster
- Contact type categorization
- Date and time stamping
- Notes and follow-up reminders
- Search contact history
- Export for documentation
- Quick-add from student list
