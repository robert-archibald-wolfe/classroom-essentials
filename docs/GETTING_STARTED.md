# Getting Started with Classroom Essentials

This guide walks you through installing and using Classroom Essentials add-ons.

## Installation

### Option 1: Google Workspace Marketplace (Recommended)

1. Visit [Google Workspace Marketplace](https://workspace.google.com/marketplace)
2. Search for the specific add-on:
   - "Classroom Timer"
   - "Random Student Picker"
   - "Group Maker"
   - "Seating Chart Generator"
3. Click **Install**
4. Grant necessary permissions
5. Open the corresponding Google app (Slides, Sheets, etc.)
6. Look for the add-on in the **Add-ons** menu

### Option 2: Manual Installation (For Developers)

Each add-on can be manually installed for testing:

1. Open Google Slides or Sheets
2. Go to **Extensions → Apps Script**
3. Copy the code from the add-on's directory
4. Save the project
5. Reload your document

## Using Classroom Timer

### Quick Start
1. Open any Google Slides presentation
2. Click **Add-ons → Classroom Timer → Start Timer**
3. Choose a preset (5, 10, or 15 minutes) OR enter custom time
4. Click **Start** - timer appears in sidebar
5. Present your slides - timer continues running

### Tips
- Keep sidebar visible while presenting
- Use pause for Q&A sessions
- Enable sound alerts for transitions
- Add 30 seconds buffer for activity setup

### Common Use Cases
- **Bell Ringers**: 5-minute timer
- **Group Work**: 10-15 minute timer
- **Presentations**: Custom timer per student
- **Test Taking**: Exact time allocations

## Troubleshooting

### Timer not appearing
- Check that add-on is installed
- Reload the presentation
- Look under **Add-ons → Manage add-ons**

### Permissions error
- Review required permissions during install
- Add-ons need access to show sidebar
- No access to your files or data

### Sound not playing
- Check browser allows audio
- Verify "Play sound" checkbox is enabled
- Some browsers block audio on first load
