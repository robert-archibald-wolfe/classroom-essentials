# Station Rotation Timer - Google Slides Add-on

Multi-timer for managing rotating stations.

**Status:** Coming Soon

## Features (Planned)

- Set up multiple stations with individual times
- Visual display of all station timers
- Automatic rotation alerts
- Transition time between stations
- Current station highlighting
- Audio/visual cues for rotation
- Save station configurations
