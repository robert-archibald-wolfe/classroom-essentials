# Task Checklist - Google Slides Add-on

Student-facing checklist displayed during presentations.

**Status:** Coming Soon

## Features (Planned)

- Create task checklists for activities
- Display checklist overlay on slides
- Students can mentally track progress
- Customizable task items
- Save checklist templates
- Timer integration per task (optional)
